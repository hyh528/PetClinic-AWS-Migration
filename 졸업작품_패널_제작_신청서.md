# 졸업 작품 패널 제작 신청서

## 작품 개요
○ 이 프로젝트는 널리 알려진 오픈소스 애플리케이션인 Spring PetClinic Microservices를 기존의 온프레미스 환경에서 AWS 클라우드 네이티브 환경으로 성공적으로 이전(Migration)하는 과정을 담은 포트폴리오 프로젝트입니다.
○ 시스템은 고객 서비스, 수의사 서비스, 방문 서비스, 관리 서버 등 독립적인 마이크로서비스로 구성되며, API Gateway를 통해 통합되고, 데이터베이스, 모니터링, 프론트엔드 배포까지 포함한 완전한 클라우드 네이티브 애플리케이션입니다.

## 제작 목적
○ 마이크로서비스 아키텍처의 설계 및 구현 능력을 향상시키고, 실무에서 요구되는 IaC 도구(Terraform)를 활용한 클라우드 인프라 자동화 기술을 학습하기 위함입니다.  
○ 졸업 작품으로서 현대적인 소프트웨어 개발 방법론(DevOps, CI/CD)을 적용하여, 확장 가능하고 유지보수 가능한 시스템 구축 경험을 쌓는 것을 목표로 합니다.

## 작동 원리
○ 각 마이크로서비스는 Spring Boot로 개발되어 독립적으로 실행되며, Eureka를 통한 서비스 디스커버리와 API Gateway를 통한 라우팅으로 통합됩니다.  
○ Terraform 레이어를 순차적으로 실행하여 AWS 리소스(VPC, ECS, RDS, Lambda 등)를 프로비저닝하고, Docker 컨테이너로 패키징된 서비스를 배포합니다. GenAI Lambda는 Amazon Bedrock을 활용한 AI 기능을 제공합니다.

## 작품 사진(고해상도)
○ 고해상도 작품 사진 파일은 별도로 제출 예정입니다. (시스템 아키텍처 다이어그램, 배포 화면, 애플리케이션 실행 화면 포함)
